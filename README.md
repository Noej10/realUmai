+ 맛집 등록
  + 소개하고 싶은 맛집을 등록
  + 정확한 위치를 주소로 입력해 공유

+ 맛집 확인
  + 맛집에 대한 평가를 댓글 형식으로 등록
  + 무엇을 작성할 지 고민하지 않게 항목 별로 점수를 책정할 수 있는 기능
